Reversing 🧬 Injection 💉 Game Code 💘 RPG Project ⚔️ SDK Manager 🍡 Dev Tools 🔍 Game Info 🌮 Backups 🧺

The tools in Reversing focus on game code.

💉 Injection lets you insert your own code into the game at run-time.

💘 Game Code views and deobfuscates the game's run-time code.

🍡 SDK Manager lets you manage NW.js SDK versions.

🔍 Dev Tools enables additional tools in the game.

🌮 Game Info shows and edits information about the game.

🧺 Backups lets you backup and restore the game.
