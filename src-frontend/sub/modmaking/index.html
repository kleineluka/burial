The tools in Mod Making streamline TCOAAL modding.

⚔️ Bundler generates RPG Maker MV projects and mods.

♻️ Converter makes mods compatible with the Tomb modloader.

🪄 Decompiler turns mods into editable RPG Maker MV projects.

🍱 Mod JSON lets you create and edit mod metadata.

☁️ Repo JSON prepares your mod to be published.

♊ Differences compares two versions of a mod.
