The tools in Mod Manager install and manage mods.

🍱 Mod Packs are an easy one-click setup for modded TCOAAL

🪦 Mod Loader sets up the Tomb modloader for you.

🕹️ Mod Browser lets you search and install mods.

🛍️ Installed Mods manages your installed mods.

🎮 Profiles let you launch different sets of mods.

📥 External is for downloading mods from third-party websites.

📰 Upcoming features mods that haven't released yet.
