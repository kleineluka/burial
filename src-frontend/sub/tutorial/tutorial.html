Burial has many features, all stored in the sidebar.

1. Each option will take you to its own page, with its own sidebar.

2. You can minimize the sidebar in the top left to get more screen space!

3. You can also return home at any time in the top left.

You might just want the Mod Manager.

1. This will let you easily manage fan-created mods. You can browse, install, remove, or update.

2. Make sure you have a mod loader installed first. You can automatically install this in the Mod Manager tab as well!

3. If you don't care about creating content, and just want to play the game with mods, this is all you need!

For creators - there's a lot to explore.

1. Each section page has its own description of what it does.

2. There's also a lot of documentation on the Github page to use!

3. Names are also self-explanatory! For example, the Resources section contains tools to decrypt assets from the game.

And you can always come back!

1. The Settings page has a shortcut to restart the tutorial.

2. You can also change your TCOAAL path there.

3. Feel free to reach out on Github or Discord if you need help!
