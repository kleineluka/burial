The tools in Resources focus on getting assets from TCOAAL.

🔪 Decryption: Restore .k9a files to their original format.

🩹 Encryption: Encrypt files to .k9a format.

🫧 Sifter: Sort and export specific categories of assets.

🥺 Sprites: Create sprite templates from game assets.
